# 厦门航空国内客票退票政策（多源知识库）

## 适用范围
适用于厦门航空实际承运的国内航班普通客票，特殊产品以对应产品规则为准。

## 自愿退票手续费
- 起飞前168小时（含）以前：经济舱5%-20%，公务舱0%-10%。
- 起飞前48-168小时：经济舱30%-40%，公务舱10%-20%。
- 起飞前4-48小时：经济舱50%-60%，公务舱30%-40%。
- 起飞前4小时以内及起飞后：经济舱70%-90%，公务舱40%-60%。

## 特殊减免
- 婴儿票可免收退票费。
- 因病退票提供二级以上医院在航班起飞前出具的诊断证明，可免收退票费；同行人（限2名）同时办理可免手续费。

## 退款时效
退票款一般退回原支付账户，通常为7-15个工作日。
