# 厦门航空国内客票改签政策（多源知识库）

## 自愿变更办理时限
航班起飞前可以申请变更，但须在原航班起飞前取消原座位，否则按误机处理。

## 自愿变更手续费
- 起飞前168小时（含）以前：经济舱5%-10%，公务舱0%-5%。
- 起飞前48-168小时：经济舱20%-30%，公务舱5%-15%。
- 起飞前4-48小时：经济舱40%-50%，公务舱20%-30%。
- 起飞前4小时以内及起飞后：经济舱60%-70%，公务舱30%-50%。

## 补差价规则
- 新票价高于原票价：补足差额。
- 新票价低于原票价：差额不退，通常需按自愿退票后重购。

## 签转规则
仅限可签转票价且新承运人与厦航有互开协议；否则按自愿退票处理。
